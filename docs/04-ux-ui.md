# 04 — UX/UI: макеты основных экранов

Низкодетальные wireframes (ASCII) ключевых экранов с заметками по поведению.
Дизайн-система: Tailwind + shadcn/ui, 8px-сетка, тёмная/светлая тема, полноценный
**RTL** для арабского (зеркалирование раскладки, иконок направления, отступов).
Все подписи — из языковых ресурсов (см. `05-i18n.md`).

Общие принципы:
- Левый сайдбар-навигация (в RTL — справа), верхняя панель с поиском, переключателем
  языка, уведомлениями и аватаром.
- Адаптивность: на мобильном сайдбар сворачивается в нижнюю таб-навигацию.
- Состояния: skeleton-загрузка, пустые состояния с CTA, ошибки с retry.

│ 11:00  ░░░░    ░░░░    ░░░░     ▣Max    ░░░░                                │
│ ...                                                                        │
│ Клик по ▣ → детали урока (войти/перенести/отменить)                        │

## Дизайн-токены (ориентир)
- Шрифт: Inter (LTR) + системный арабский (Noto Sans Arabic) для `ar`.
- Радиусы: 8/12px; тени мягкие; основной акцент — синий 600; успех — зелёный 600.
- Доступность: контраст AA, фокус-обводки, навигация с клавиатуры, ARIA-метки.
